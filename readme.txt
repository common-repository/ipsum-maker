Ipsum Maker lets you create your own themed lorum ipsum text generator

== Description ==

I constantly use lorum ipsum text when creating sites. I'll bet you do, too. With Ipsum Maker, now you can have your very own lorum ipsum text based on any theme you choose. For an example, check out http://durdenipsum.com/

== Installation ==

1. Upload the `ipsum-maker` folder to your `wp-content/plugins/` directory. 
1. Activate the plugin.
1. Add Ipsum entries from the WP dashboard.
1. Add the `[ipsum]` shortcode to a page or post.
